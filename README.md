# DayBrain — Free Daily Planning Template

A focused daily planning template for solo professionals, freelancers and founders. Built around the 5-minute planning framework.

**[→ Open the interactive template](https://gmhsource-lab.github.io/DayBrain-5-Minute-Daily-Planner-Template/)**

---

## What's inside

- **Non-Negotiables** — daily commitments checked off before everything else
- **Top 3 Priorities** — ranked by what actually moves the needle
- **Time Blocks** — 7am to 6pm, protect your deep work
- **Brain Dump** — clear your head before you start
- **Not Right Now** — parking lot for mid-day interruptions
- **Evening Review** — wins, misses, tomorrow's first move
